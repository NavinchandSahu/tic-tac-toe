## How to Play
1. Run the game file (`python tic_tac_toe.py`)
2. Player X starts the match
3. Click a tile to place your symbol
4. Game ends when:
   - A player aligns 3 marks in a row, OR  
   - The board fills with no winner (tie)
5. Click **Restart** to play again
